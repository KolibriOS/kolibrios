# Параметры запуска sdlquake (KolibriOS)

Этот список отражает реальные параметры, доступные в текущей сборке из этого репозитория.
Проверено по `Makefile.kos` и файлам из `C_SRC`.

## Параметры (по блокам)

### Файлы и данные
- `-basedir <dir>` - задать базовый каталог игры.
- `-cachedir <dir>` - каталог кеша (если далее идёт аргумент с `-`, кеш выключается).
- `-game <gamedir>` - добавить `basedir/gamedir` как override-данные.
- `-path <p1> <p2> ...` - задать поисковые пути полностью.
- `-hipnotic` - mission pack Hipnotic.
- `-rogue` - mission pack Rogue.
- `-proghack` - режим совместимости для `progs.dat`.

### Сеть и сервер
- `-dedicated [maxclients]` - выделенный сервер.
- `-listen [maxclients]` - слушать как сервер.
- `-port <p>` / `-udpport <p>` / `-ipxport <p>` - сетевой порт.
- `-record` - записать VCR (`quake.vcr`).
- `-playback` - воспроизвести VCR (`quake.vcr`).

### Видео и ввод
- `-winsize <width> <height>` - размер окна/буфера SDL в пикселях. **Используется лаунчером.**
- `-fullscreen` - полноэкранный (безрамочный) режим. **Используется лаунчером** вместе с `-winsize` под размер экрана.
- `-nomouse` - выключить инициализацию мыши.

### Звук
- `-nosound` - выключить звук.
- `-simsound` - эмуляция звука без реального вывода.

### Производительность, память и диагностика
- `-surfcachesize <kb>` - размер surface cache в KB.
- `-particles <n>` - число частиц.
- `-zone <kb>` - размер zone memory в KB.
- `-minmemory` - минимизировать расход памяти.
- `-condebug` - писать `qconsole.log` в текущий `com_gamedir`.
- `-safe` - безопасный режим: включает набор «-no*» флагов. На KolibriOS реально действуют `-nosound` и `-nomouse`, остальные относятся к другим платформам.

Размер кучи (8 МБ) автоматически увеличивается под выбранное разрешение в `sys_sdl.c`, чтобы видеобуферы (z-buffer + surface cache) не переполняли её на больших окнах/полноэкранном режиме.

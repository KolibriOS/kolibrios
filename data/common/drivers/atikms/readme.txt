
Поддерживаются дискретные и интегрированные чипсеты
семейств ATI R100-R600, Evergreen, Northern Islands, Southern Islands 

Драйвер состоит из двух частей: загрузчика atikms и драйвера ядра
atikms.dll. Оба файла обязательно должны находиться в одной папке.

Запуск драйвера: <путь>/atikms <командная строка>

Параметры командной строки:
 -l <полный путь к лог-файлу> Файл должен быть доступен для записи.
    по умолчанию /tmp1/1/atikms.log 

 -m <ширина x высота х частота> установить видережим
    если частота не указана то принимается 60 Гц
    Ядро Корибри не поддерживает разрешение более 2.6M пикселей!

-n  отключить определение видеорежимов и kms

Примеры:
 -m 1280х1024х75 - установить видеорежим 1280х1024 и частотой смены кадров 75 Гц
 -m 800x600 - установить видеорежим 800х600 и частотой смены кадров 60 Гц  

Для запуска из autorun.dat командная строка не должна содержать пробелов. 

Пример:
/SYS/DRIVERS/ATIKMS -m1024x768x75-l/hd0/2/atikms.log 1    # 



Особенности реализации.

При первом запуске драйвер гасит экран два раза.

Atikms предназначен только для загрузки драйвера и установки 
видеорежима при первом запуске. Для смены видеорежима в 
процессе работы используйте vmode или vmode <видеорежим> 
Например vmode 1152x864х75

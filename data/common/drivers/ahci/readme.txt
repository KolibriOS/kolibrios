Драйвер для AHCI

Работает поиск контроллеров и определение подключённых АТА дисков. Если драйвер отработает нормально на доске отладки будет модель устройства и количество секторов.

В драйвере AHCI используется свой собственный кэш с поиском по хеш-таблице, за счет чего чтение работает в несколько раз быстрее, чем через биос (пункт b на голубом экране).

ПРЕДУПРЕЖДЕНИЕ: не рекомендуется использовать данный драйвер когда включен пункт 'b' на голубом экране (чтение дисков через биос)

# Palitra v0.8

Подбор цвета и заливка фона рабочего стола.

Программу можно запускать с параметрами. Первый параметр - режим (`N`, `H`,
`B`, `D`), далее - до двух цветов в hex (основной и дополнительный).

## N - цвета по умолчанию

Стартовые цвета берутся из параметров.

```
/sys/media/palitra N 00FF00FF 00FF0000
/sys/media/palitra N 00F724A3 00736698
```

## H - заливка фона градиентом (hidden)

Окно не отрисовывается: фон рабочего стола заливается градиентом из двух
переданных цветов, после чего программа завершается.

```
/sys/media/palitra H 00FF00FF 00FF0000
/sys/media/palitra H 00F724A3 00736698
```

## B - заливка фона шумом (hidden)

То же, что `H`, но фон заливается зашумлённой заливкой на основе цвета.

```
/sys/media/palitra B 00FF00FF 00FF0000
```

## D - режим диалога выбора цвета

Программа запускается как диалог выбора цвета.

```
/sys/media/palitra D
```

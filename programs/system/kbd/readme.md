## Примечание (Heavyiron, 24.10.2006):

Владельцы данных чипсетов могут (при желании) поместить эту программу в автозагрузку.
Для этого в autorun.dat необходимо добавить, например, такую строку:
```dat
/SYS/KBD   BOOT    20      # Enable Bus Disconnect for AMD K7 processors
```
не забыв при этом увеличить количество запускаемых программ на одну в самом начале autorun.dat.

## Изменения в версии 2.0

- Код переписан: разделение кода и данных, компактная таблица детекции (8 байт на запись)
- Исправлено сканирование PCI: теперь перебираются все устройства и функции, а не только типовая позиция моста
- Исправлен nForce2: добавлен обход ошибки C1-HALT (0x6E=1), без которого включение могло вешать систему
- Исправлен SiS 741: использовался неверный регистр (0x6A вместо 0x6C), что могло приводить к зависанию
- Уточнены маски регистров (nForce, VIA, SiS 735) по даташитам и athcool — удалены лишние биты
- Исправлена таблица детекции: KLE133 (правильный ID 0x3112), добавлены KX133 и SiS 755, удалён несуществующий AMD 751S
